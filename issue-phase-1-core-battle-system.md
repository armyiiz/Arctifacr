# เฟส 1: สร้างระบบการต่อสู้หลัก (Core Battle System)

**Objective:** พัฒนาระบบการต่อสู้ที่เป็นหัวใจหลักของเกม

**Tasks:**

- [ ] สร้างหน้าจอการต่อสู้ (Battle Screen) ที่มีกระดานสำหรับผู้เล่นและ AI ฝั่งละ 4 ช่อง
- [ ] สร้างคอมโพเนนต์การ์ด (Card Component) ที่สามารถแสดงหน้า-หลัง และตัวเลขได้
- [ ] พัฒนากลไกการจั่วการ์ด, การนำการ์ดขึ้นมือ (Hand) และการลากการ์ดไปวางบนกระดาน
- [ ] สร้าง Logic สำหรับการตัดสินผลแพ้-ชนะในแต่ละช่อง, การคำนวณดาเมจ, และการลด HP
- [ ] จำลองการต่อสู้ 1 รอบให้สมบูรณ์ โดยใช้ข้อมูลเด็คและศัตรูแบบง่ายๆ ไปก่อน
