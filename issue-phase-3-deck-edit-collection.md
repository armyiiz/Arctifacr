# เฟส 3: สร้างหน้า Deck Edit และ Collection

**Objective:** ให้ผู้เล่นสามารถจัดการเด็คและดูการ์ดที่สะสมได้

**Tasks:**

- [ ] สร้างหน้า Collection เพื่อแสดงการ์ดทั้งหมดที่ผู้เล่นมี
- [ ] สร้างหน้า Deck Edit ที่ผู้เล่นสามารถจัดเด็ค 12 ใบตามกฎ และบันทึกเด็คได้
- [ ] เชื่อมเด็คที่ผู้เล่นจัดไว้เข้ากับระบบการต่อสู้
- [ ] พัฒนาระบบรับการ์ดใหม่เข้า Collection จากการชนะศัตรูหรือเปิดหีบ
